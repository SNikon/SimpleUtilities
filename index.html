Simple Utilities Project.

This project contains code snippets and modules for small utilities that can be used in environments that have limited resources. This is not intended to be a professional library that competes with other frameworks for development, but it may come in handy in particular situations where libraries such as jQuery or Angular must be avoided but to company constraints, restrictive environments, or developing on top of problematic implementations of ECMAScript.

This is a work-in-progress, extending as I see the need to add new features for my own use, and I am sure that there will be much better implementations to everything that I can possibly write, as I do not have the time or resources to evolve this into a mature library. I will try commenting everything and keep the code as clean as I can, and feel free to judge the code for yourselves, comment on it, or help with this library!

Module Summary (WIP)

Debug

Small module to handle logging information. Provides the means to turn logging on and off at any time and allows attaching to a DOM element to write directly into it, for visual feedback where a console is not available.

Dom

TODO Domain manipulation. At this time this module is only used to get computed styles.

Google_Analytics

TODO Sets up google analytics bindings and allow for quick methods to track what you need in the manner that you need. This module is in queue to be finished.

Obj_Iterator

Single purpose module, allows safe iteration through objects and allows adding a condition to iterate if necessary. This module will be merged into a larger module with more object manipulation.

Scroll

Allows manipulating the scrolling of a div using ECMAScript. I will extend this to allow for fully circular scrolling and most likely automated one.

String

String manipulation. Allows serializing an object into several forms such as form-data, query string and JSON string. Yes, one would assume that window.JSON exists everywhere for stringification but that is not true unfortunately. Also contains '.format' method that acts like the amazing C# String.Format, a slightly different syntax that of the marvelous printf in C and C++.

Template

An absolutely unnecessary module if you use RequireJS, which allows for loading and caching HTML templates. You can find the alternative to this in the project Tech-Experiments in @SNikon that uses RequireJS and the require-text plugin that will potentially suit your needs much better.

Xhr

A module that is absolutely necessary for me but probably irrelevant to you. Allows similar functionalities to jQuery's ajax calls, and also allows adding events after the request has already being sent and possibly received. The module returns an object containing the information to be submitted and, if concluded, the information received. This module makes no promise that if you insert a "complete" event before a "success" or "error" event, that it will be triggered in that order.
